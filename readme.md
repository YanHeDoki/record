# 最糟糕的一年
2025年应该是工作到目前为止最糟糕的一年。
本来我是不喜欢做日记或者是什么记录的，但是2025年发生了太多,我不知道后续会不会更糟糕，但是肉眼可见明年至少不会更差了
实际上我不知道记录些什么，但是去年这个时候我还在一个比较轻松的环境准备离职找下一份工作，如果当时的我记得会发生一些什么的话，或许会重新审视一下计划至少会换城市
来到广东或者说深圳就是一个最大的错误，一切痛苦的根源就是从这里开始，来了之后只有无穷无尽的加班,和吃不完的饼和坎坷的求职
两份都不怎么样的工作还导致了简历的不好看，不知道后面再次找工作的时候要怎么解释，也可能不知道还没有机会再找开发的工作
还有到处都是人挤人，从早上到通勤到加班结束竟然地铁都是满人，甚至没有非机动车道，来到深圳之前难以想象道路建设可以做到这么差劲，人，电动车，自行车混在一起
我不注意就会骑到机动车道上，甚至还有罚单
